# Ticket Tracker

Internal ticket management app.

- Backend: FastAPI + SQLite — see [`backend/README.md`](./backend/README.md)
- Frontend: React + TypeScript — see [`frontend/README.md`](./frontend/README.md)

## Quick start

1. Start the backend (see backend README) — runs on `http://127.0.0.1:8000`
2. Start the frontend (see frontend README) — runs on `http://localhost:5173`
